# Contributing to Autonomous Task Agent

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR-USERNAME/autonomous-task-agent.git`
3. Follow the setup in README.md
4. Create a branch: `git checkout -b feature/your-feature-name`

## Pull Request Guidelines

- One feature / fix per PR
- Include a description of what changed and why
- Screenshots for UI changes
- Test manually before submitting

## Code Style

- JavaScript: ES modules, async/await, descriptive variable names
- CSS: BEM-ish naming, CSS variables for all design tokens
- SQL: UPPERCASE keywords, snake_case identifiers

## Reporting Issues

Use GitHub Issues. Include:
- Steps to reproduce
- Expected vs actual behavior
- Browser and OS
- Console errors (screenshot or paste)

## Questions?

Open a GitHub Discussion or tag your question with the `question` label in Issues.
